## Requirement
- Laravel 11
- PHP 8.3 or Above
  
## Installlation
1. **Clone This Project**
2.   #### `composer install`
3. #### `php artisan key:generate`
4. **Setup .env**
5. #### `php artisan migrate`
6. #### `php artisan serve`
7. Now You can run the app
